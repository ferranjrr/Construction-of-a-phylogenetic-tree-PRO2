Descripció Detallada

Representa una espècie.

Cada espècie té associat dos strings, un identificador id i un gen. Els caràcters del string gen poden ser només 'A', 'C', 'T' i/o 'G'. Es pot calcular la distància entre l'espècie del p.i. i una altra. L'espècie té associada una seqüència kmer (que pot estar no calculada, per tant buida) que serveix per calcular les distàncies entre espècies. kmer conté uns certs strings i és generada a través del gen de l'espècie i l'enter k que indica el criteri de càlcul del Cjt_Especies el qual aquestes pertanyen. La classe té operacions de lectura i escriptura.

map<string,int> Especie::kmer
private

Seqüència kmer

Conjunt de les subseqüències contínues (del gen) de longitud k >= 1 (criteri de càlcul donat); o també anomenat els k-mers de gen. Si n és la longitud de gen, els kmer és el conjunt: {g[i..j + k − 1] | 0 ≤ i < n + 1 − k}, on g[i..j] denota la subseqüència de gen entre els símbols i-èssim i j-èssim de gen, que assumim que s'indexen de 0 a n-1.

Les claus (strings) són les sucessions de k caràcters del gen. L'enter > 0, és el número de cops que apareix cada element del conjunt.
